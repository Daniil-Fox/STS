Разработка Технологических Решений для очистных сооружений

Для выработки решения, гарантирующего достижение требуемых параметров очищенных сточных вод выполняются следующие работы:

  • Подготовка, совместно с Заказчиком, Технического задания на разработку технологического процесса очистки сточных вод;
  • изучение, инвентаризация, классификация источников образования загрязнений в результате производственных процессов предприятия — производственных, хозяйственно-бытовых и ливневых стоков;
  • отбор проб, проведение лабораторного анализа сточных вод;
  • разработка принципиального технологического решения для ОС;
  • подтверждение в лабораторных условиях выбранного технологического решения по очистке стоков;
  • согласование с Заказчиком, подтверждённого испытаниями, технологического решения способа очистки;
  • определение принципиального состава технологического оборудования;
  • утверждение Заказчиком технологической схемы очистки сточных вод;
  • согласование Заказчиком предварительной оценки стоимости решения.
